# Fiora-App

这是 Fiora(https://fiora.suisuijiang.com/) 的APP版, 基于 react-native 和 expo, 支持安卓和iOS

## Android

点击链接或者扫码下载APK

## iOS

1. Apple Store 搜索 expo 并下载
2. 打开 expo, 并使用碎碎酱的账号登录(私聊要账号)
3. 登录成功后点击 fiora 启动应用
